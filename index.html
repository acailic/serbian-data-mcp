Prompts & Results
What you can ask
Real examples of prompts that work with the server. Paste any of these into Claude, Cursor, or any MCP client.

🔍 Find Population Data

search
"Search for population census data from the Serbian statistical office"
Returns {datasets: [{id, title, organization}], total: 42} — 42 matching datasets from РЗС with resource IDs ready for download.

📊 Visualize Air Quality

visualization
"Search for air quality data, download it, create a line chart of PM10 levels over time in Niš"
Auto-executes: search → get_resource_data → data_profile → create_chart(line) — returns interactive Plotly chart with hover tooltips.

🇷🇸 Serbia Population Map

map
"Create a choropleth map of Serbia showing population density by district"
Renders 25 okrug boundaries from Natural Earth data. Returns {filepath: "exports/serbia_map.html"} — self-contained interactive HTML map.

📈 Budget Analysis

analysis
"Find budget execution data, extract insights showing the biggest spending changes year-over-year"
Auto-executes: search → download → data_profile → extract_data_insights — returns severity-ranked findings like "Defense spending +23% YoY".

📰 Data Journalism Story

story
"Create a data story about Serbia's population decline — find data, show trends, generate an infographic"
Auto-executes full pipeline: search → download → insights → chart → infographic — returns {filepath, headline: "-12% decline", big_number: "1.4M"}.

📉 Forecast Future Trends

forecast
"Download population data and forecast what the population will be in 2035 at the current rate"
Runs forecast_data(method='linear', periods_ahead=10) — returns {projection_note: "At -0.4%/yr, Serbia will have ~5.8M by 2035", r_squared: 0.97}.

🏛️ Real Estate Prices

search
"Pronađi podatke o cenama stanova u Beogradu i napravi bar chart po opštinama"
Serbian queries work natively. Returns Belgrade real estate datasets with XLSX resources. Creates per-municipality price comparison chart.

🔗 Compare Two Datasets

compare
"Compare these two population datasets: id-A and id-B. Which has better data quality and more recent updates?"
Returns {same_organization: false, shared_formats: ["xlsx"], quality_score: 0.85} — side-by-side comparison with recommendations.

Installation Options
Works everywhere
Install on any MCP-compatible client. No API keys needed.

🐍 PyPI (Recommended)

Install globally and add to any MCP client configuration.
pip install serbian-data-mcp

🟣 Claude Desktop

Add to your Claude Desktop config file.
{
  "mcpServers": {
    "serbian-data": {
      "command": "serbian-data-mcp"
    }
  }
}

📦 Smithery

Install via the Smithery CLI — auto-configures your client.
smithery mcp add acailic/serbian-data-mcp --client claude

⬡ Cursor

Add to Cursor's MCP settings.
{
  "mcpServers": {
    "serbian-data": {
      "command": "serbian-data-mcp"
    }
  }
}

🔧 uvx (No Install)

Run directly without installing — good for testing.
{
  "mcpServers": {
    "serbian-data": {
      "command": "uvx",
      "args": ["serbian-data-mcp"]
    }
  }
}

💻 From Source

Clone and run for development or customization.
git clone https://github.com/acailic/serbian-data-mcp
cd serbian-data-mcp
uv sync
uv run python -m serbian_data_mcp
Recommended Workflow
7 steps from question to answer
The LLM resource guide teaches this automatically. But here's the pattern:
MCP Tool Pipeline 
1. search_datasets("kvalitet vazduha")        ← Find datasets
2. get_dataset("id-123", detail_level="metadata") ← See resources
3. get_resource_data("res-456")            ← Download data
4. data_profile(data)                      ← Learn columns
5. transform_data(data, "filter", ...)        ← Clean data
6. create_chart(data, "line", ...)            ← Visualize
7. export_visualization(figure, "html")    ← Save file
