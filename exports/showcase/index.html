    🎯 Summary & Recommendations

    Ready for Production (Start Here)

    • Air Quality Dashboard — 14 years, 40+ stations, daily data. Highest ROI dataset on the portal. Build animated time-series with city toggle.
    • Employment Trends Dashboard — 10 years of monthly data. Line charts with forecasting to 2030. Seasonal decomposition.
    • Census Comparison Map — Choropleth of household change 2011→2022. Slope chart for ranking changes. Infographic with demographic insights.

    Worth Exploring (Moderate Effort)

    • Municipal Budget Analysis — Requires per-municipality XLSX parsing (varied structures). Sankey diagrams of budget flows. Comparison across municipalities.
    • Real Estate Price Map — Tax assessment prices as baseline. Choropleth by municipality. Year-over-year trend analysis.
    • Cross-Dataset Insights — Employment × demographics, air quality × budgets, registries × economic data.

    Technical Improvements Needed

    • Fix resource ID lookup — The `_find_resource` method only scans 150 recent datasets. Must search by parent dataset ID first.
    • UTF-8 BOM handling — RZS JSON endpoint returns BOM, parser must use `utf-8-sig`.
    • Search language — Portal API only supports Cyrillic. The bilingual intelligent search is essential for usability.
    • Rate limiting — With 3,443 datasets, bulk operations need pagination and caching.
